ОПИСАНИЕ ПРОЕКТА

Проект представляет собой Telegram-бота.
В процессе работы с ботом пользователю задаются вопросы, на которые необходимо ответить, выбрав один из доступных вариантов.
На каждый ответ бот отправляет реакцию в виде оценки правильности ответа. Если ответ неверен, указывается правильный вариант.
По результатам ответов формируется мини-отчёт: количество верных ответов и процент правильных решений.
В любой момент времени есть возможность завершить прохождение викторины.
По окончании викторины предоставляется возможность пройти её повторно.
В течение всего процесса прохождения викторины пользователь может запросить текущий результат, после чего получает мини-отчёт о своей успеваемости.

ПОРЯДОК ЗАПУСКА ПРОЕКТА

1. Откройте Telegram
2. В поиске напишите @BotFather
3. Выберите официального BotFather (с синей галочкой ✓)
4. Отправьте команду BotFather: /newbot
5. Придумайте имя для бота. Например: My_Test_Bot
6. Придумайте username (должен заканчиваться на bot)
7. После успешного создания BotFather отправит вам сообщение с токеном: 1234567890:ABCdefGHIjklMnOpQRstUVwxyZ-abcdeFGHIJ....
8. Добавьте токен в переменную TELEGRAM_BOT_TOKEN="" в файле проекта .env
9. Запускаем проект go run main.go
10. Ищем в поиске telegram, бота по имени которое мы задали.
